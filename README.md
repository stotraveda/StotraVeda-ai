# 📿 StotraVeda AI Dashboard

Welcome to **StotraVeda AI**, a comprehensive, full-stack responsive web hub tailored to bring ancient Vedic wisdom into the modern digital landscape. Powered by high-speed inference engines and real-time semantic discovery, this system features a curated multi-agent workspace housing **20 specialized AI engines** dedicated to Sanatana Dharma.

---

## 🌌 Core Features & Agent Suite

The platform utilizes a dynamic system context matrix that transforms the chat experience based on your chosen track:

* **🛕 Temple Guide AI:** Discover structural architecture, traditional lineages, and historical metrics of ancient mandirs.
* **📅 Panchang AI:** Delve into astronomical lunar calculations, Tithis, Nakshatras, and solar transits.
* **⭐ Muhurat AI:** Isolate highly auspicious times while filtering out negative phases like Rahu Kaal.
* **📿 Stotra & 📖 Mantra AI:** Access Sanskrit verses backed with word-by-word structural breakdowns and acoustic pronunciation phonetics.
* **🔥 Homam & 🙏 Puja AI:** Review precise ritual materials (Samagri), layout dimensions, and step-by-step Shodashopachara setup guides.
* **🏡 Vastu Shastra AI:** Evaluate directional elements and cosmic spatial orientation geometries.
* *And 11 other custom engines tailored for Ayurveda, Scripture study, Pilgrimages, Baby naming, and automated Document Summarization.*
